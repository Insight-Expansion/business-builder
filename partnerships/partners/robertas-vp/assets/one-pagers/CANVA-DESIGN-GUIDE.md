## Step 1: Generate QR Code (5 minutes)

1. Go to **https://www.qr-code-generator.com/**
2. Enter URL: `https://www.nexrizen.com/contact`
3. Settings:
   - Frame: None (clean)
   - Foreground color: `#502cff` (or black for simplicity)
   - Background: White
   - Size: 300x300px minimum
4. Download as PNG
5. Save to: `/partnerships/partners/robertas-vp/assets/brand/nexrizen-contact-qr.png`

## Step 2: Set Up Canva (10 minutes)

### Create New Design
1. Go to canva.com
2. Click "Create a design"
3. Select "Custom size"
4. Enter dimensions: **210 x 297 mm** (A4)
5. Click "Create new design"

### Upload Brand Assets
1. Click "Uploads" in left sidebar
2. Upload these files:
   - `nexrizen-logo.png` (from assets/brand/)
   - `nexrizen-contact-qr.png` (QR code you just created)
   - Optional: Founder photo from website if desired

### Set Up Brand Colors
1. In color picker, add these brand colors:
   - Primary: `#502cff` (purple/blue accent)
   - Navy: `#1e293b` (headings)
   - Slate: `#475569` (body text)
   - Blue: `#2563eb` (highlights)
   - Light bg: `#f8fafc` or `#eff6ff`
   - White: `#ffffff`

## Design Best Practices

### Typography
- **Headings:** Inter Bold, 16-20pt, `#1e293b`
- **Subheadings:** Inter SemiBold, 12-14pt, `#2563eb`
- **Body:** Inter Regular, 9-11pt, `#475569`
- **Line height:** 1.4-1.6 for readability

### Colors
- Use `#502cff` sparingly (CTAs, accents only)
- Rely on navy/slate for text
- Light blue/white backgrounds
- Avoid more than 3 colors per section

### Spacing
- **Margins:** 15mm on all sides
- **Section spacing:** 10-12mm between sections
- **Line spacing:** 1.5 for body text
- **White space:** Don't crowd - let design breathe

### Visual Elements
- Rounded corners: 3-5mm radius
- Subtle shadows: 10-20% opacity
- Icons: Simple, monochrome or gradient
- No stock photos (abstract visuals or icons only)
