## How to Use This Library

### For Sales Conversations
1. **Identify client vertical** (Legal? Construction? Sales enablement?)
2. **Open relevant vertical file** (e.g., `Legal_Tech.md`)
3. **Reference specific case studies** that match their pain points
4. **Use metrics** to build credibility (time savings, cost reduction, etc.)

### For Proposals
1. **Include 2-3 relevant case studies** from client's vertical
2. **Add 1 cross-vertical case** showing technical depth
3. **Reference internal tools** (Proposal Builder, 3-Day AI) to show innovation

### For Marketing
1. **Website case studies** - Use public-facing versions (authorization needed)
2. **Vertical-specific landing pages** - Pull from relevant vertical file
3. **Blog posts** - Deep dive into individual case studies
