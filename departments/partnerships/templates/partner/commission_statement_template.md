## TEMPLATE USAGE NOTES

**How to use this template:**

1. **Monthly preparation** (by 5th of following month)
   - Pull all client payments received during previous month
   - Filter for partner-attributed clients (check CRM/attribution log)
   - Calculate 15% commission on each payment
   - Prepare payment (wire transfer, etc.)

2. **Fill in template**
   - Replace all [bracketed] fields with actual data
   - Include all relevant tables (revenue detail, active engagements, pipeline)
   - Update YTD summary
   - Add any relevant notes or updates

3. **Review and approve**
   - Finance team reviews calculations
   - Wei or authorized person approves
   - Process payment

4. **Send to partner**
   - Export as PDF (not editable)
   - Send via email with subject: "Nexrizen Commission Statement - [Month YYYY]"
   - CC accounting/finance for records

5. **File for records**
   - Save PDF in partner's commission folder
   - Log in commission tracking system
   - Update partner's PARTNER.md file with YTD totals

**Frequency:** Monthly (by 5th of following month)

**Recipients:**
- Partner (primary contact email)
- Partner assistant if applicable
- CC: Nexrizen finance/accounting for records

**Format:** PDF (professional, not editable)
